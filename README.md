# 会议室使用调度系统
# 微信小程序实现用户界面
# Vue.js+Element-UI实现后台管理系统
# Node.js+MongodDB构建项目后端
## 如何使用？
#### 1.git clone https://github.com/hubtoks/React-Blog.git   
#### 2.在serve目录下，使用npx nodemon index.js来启动项目后端，由于项目后端使用MongoDB数据库，请确保MongoDB服务已经开启
#### 3.在admin目录下，npm i安装依赖，使用npm run serve来启动后台管理系统
#### 3.在微信开发者工具中打开miniProgram目录,使用npm i安装依赖，点击重新编译
